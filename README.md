# Premium NGO Website Template

A production-ready, scalable, and SEO-optimized starter template for NGO projects.

## Features
- **7 Pages:** Home, About, Programs, Gallery, Blog, Contact, Testimonials.
- **Tech Stack:** Vanilla HTML5, CSS3 (Modular), and JS.
- **Mobile Responsive:** Fully optimized for all screen sizes.
- **SEO Ready:** Meta tags, Open Graph, Sitemap, and Schema markup included.
- **WhatsApp Integration:** Floating chat button with pre-filled messages.
- **Dynamic Blog:** Loads posts from a JSON file (no backend required).
- **Performance:** Lazy loading images and minimal JS usage.

## Folder Structure
- `/assets/css/`: Modular CSS files (main, components, animations, responsive).
- `/assets/js/`: Functional JS (main logic, form handling, WhatsApp).
- `/components/`: Reusable HTML snippets for Navbar and Footer.
- `/blog-data/`: JSON file for managing blog content.
- `/seo/`: Robots.txt, Sitemap, and Schema markup.

## How to Use
1. **Customize Content:** Update the text in HTML files and `blog-data/posts.json`.
2. **Images:** Replace Unsplash placeholder URLs with your actual project images in `/assets/images/`.
3. **Contact Form:** The form is ready for Netlify Forms or Formspree. Just add your endpoint in `contact.html`.
4. **WhatsApp:** Update the phone number in `assets/js/whatsapp.js`.
5. **Analytics:** Add your Google Analytics ID in the script section of `index.html`.
